# Overview

Sentinel AI is a personal software engineering and artificial intelligence project that aims to build an autonomous web testing agent capable of understanding web applications, generating intelligent test scenarios, executing browser interactions and reporting software defects automatically.

The project is inspired by the growing use of Artificial Intelligence in software engineering, where AI systems are beginning to assist developers by improving software quality through intelligent automation.

Although Sentinel AI focuses on autonomous software testing, the project is also a learning platform where I am developing practical skills in:

- Artificial Intelligence
- Machine Learning
- Software Engineering
- Browser Automation
- API Development
- Cloud Development
- System Design
